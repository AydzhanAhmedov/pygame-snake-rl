# pygame-snake-rl

### Used Libraries:
```
pygame
pytorch
matplotlib
```

### Hot to install ?

```sh
git clone https://github.com/AydzhanAhmedov/pygame-snake-rl.git
cd pygame-snake-rl
conda create -n env_snake_rl python=3.7
conda activate env_snake_rl
pip install pygame
pip install torch torchvision
pip install matplotlib ipython
python agent.py
```
